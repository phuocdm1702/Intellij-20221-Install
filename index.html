Hướng dẫn cài đặt IntelliJ IDEA 2021

Hướng dẫn này giúp bạn cài đặt IntelliJ IDEA 2021 trên Windows một cách dễ dàng, kèm theo hướng dẫn kích hoạt bản quyền. Các bước được minh họa bằng hình ảnh để dễ theo dõi.

Bước 1: Tải và cài đặt IntelliJ IDEA 2021

1. Tải file cài đặt

  • Truy cập link dưới đây để tải IntelliJ IDEA 2021.1.
  • Nhấn để tải file: Tại đây

2. Giải nén và cài đặt

  • Giải nén file JetBrains IntelliJ IDEA Ultimate 2021.1.zip vừa tải về.
  • Chạy file ideaIU-2021.exe.
  • Nhấn Next qua các màn hình cài đặt hoặc tùy chỉnh theo nhu cầu:
    • Nhấn Next để tiếp tục.
    • Nhấn Next hoặc tùy chỉnh theo nhu cầu cá nhân.
    • Chọn các tùy chọn như tạo shortcut, hỗ trợ 64-bit, liên kết với các loại file (.java, .py, .xml, v.v.).
    • Nhấn Install để bắt đầu cài đặt.
    • Chờ chương trình hoàn tất cài đặt.
    • Chọn Finish khi hoàn tất (không cần khởi động IntelliJ ngay).

Bước 2: Cài đặt plugin kích hoạt

1. Tải và sao chép plugin

  • Mở file vừa giải nén JetBrains IntelliJ IDEA Ultimate 2021.1.
  • Tìm file BetterIntelliJ-1.20.jar trong thư mục Licenses > JetBrains BetterIntelliJ v1.20.
  • Sao chép file BetterIntelliJ-1.20.jar vào ổ C:\ (đường dẫn: C:\BetterIntelliJ-1.20.jar).
  • Nếu xuất hiện thông báo lỗi khi dán vào ổ C:\ (như hình trên), nhấn Skip hoặc Ignore.

2. Cài đặt plugin trong IntelliJ

  • Mở IntelliJ IDEA 2021.
  • Tùy thuộc vào giao diện, thực hiện một trong hai cách sau:
  • Nếu ở màn hình chính, vào Configure > Plugins.
  • Nếu đang mở một dự án, vào File > Settings > Plugins.
  • Chọn Install Plugin from Disk.
  • Duyệt đến file BetterIntelliJ-1.20.jar trong thư mục đã giải nén và nhấn OK.
  • Nhấn OK để lưu (không khởi động lại IntelliJ nếu được hỏi).

3. Cấu hình plugin

  • Vào menu Configure hoặc Help > Edit Custom VM Options.
  • Thêm dòng sau vào cuối file:
    • -javaagent:C:/Users/ADMIN/Downloads/JetBrains IntelliJ IDEA Ultimate 2021.1/JetBrains IntelliJ IDEA Ultimate
        2021.1/Licenses/JetBrains BetterIntelliJ v1.20/BetterIntelliJ-1.20.jar
  • Bạn có thể sao chép đường dẫn của file BetterIntelliJ-1.20.jar từ thư mục đã giải nén và sửa như ví dụ trên.
  • Lưu file và khởi động lại IntelliJ.

Lưu ý

Đọc trước khi thực hiện!

An toàn: Đảm bảo tải file từ link Google Drive ở trên để tránh virus hoặc mã độc.

Sao lưu: Lưu lại mã kích hoạt ở nơi an toàn để sử dụng sau này.

Khắc phục lỗi:

  • Nếu plugin không hoạt động, kiểm tra lại đường dẫn C:\BetterIntelliJ-1.20.jar.
  • Nếu IntelliJ không nhận mã, khởi động lại IDE và thử lại.
